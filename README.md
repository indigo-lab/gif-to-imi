# gif-to-imi

このレポジトリでは <https://imi.go.jp/goi/gif> で公開されている **コアデータモデル語彙 ドラフト版** をもとに
GIF から IMI への機械可読なマッピングを提供することを目的としています。

マッピングは [@indigo-lab/jx](https://github.com/indigo-lab/jx) に基づいて記述されています。
**@indigo-lab/jx** のツールを利用して、
実際に GIF (JSON 形式) から IMI (JSON-LD 形式) に変換することが可能です。
